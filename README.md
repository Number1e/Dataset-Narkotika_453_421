# Dataset Putusan Pidana Khusus Narkotika - Pengadilan Negeri Mataram

## Deskripsi
Dataset ini berisi kumpulan putusan perkara pidana khusus narkotika dari Pengadilan Negeri Mataram. Data dikumpulkan dari direktori putusan website Mahkamah Agung Republik Indonesia. Dataset ini mencakup putusan yang dipublikasikan dalam kurun waktu tiga bulan terakhir dari tanggal pengambilan data.

## Detail Dataset
| Properti | Keterangan |
| :--- | :--- |
| **Sumber Data** | Website Mahkamah Agung Republik Indonesia |
| **Pengadilan** | Pengadilan Negeri Mataram |
| **Jenis Perkara** | Pidana Khusus Narkotika |
| **Periode Data** | 3 bulan terakhir (Agustus 2025 - Oktober 2025) |
| **Format File** | `.pdf` |
| **Jumlah File** | 53 dokumen putusan |

## Struktur Data
Setiap file PDF merupakan satu dokumen putusan lengkap yang umumnya memuat informasi sebagai berikut:
* **Nomor Putusan:** Identitas unik perkara.
* **Identitas Terdakwa:** Nama (seringkali disamarkan), umur, pekerjaan, dll.
* **Dakwaan:** Pasal-pasal yang didakwakan oleh Jaksa Penuntut Umum.
* **Tuntutan:** Hukuman yang dituntut oleh Jaksa.
* **Pertimbangan Hakim:** Analisis hukum dan fakta persidangan oleh Majelis Hakim.
* **Amar Putusan:** Vonis hukuman yang dijatuhkan (pidana penjara, denda, rehabilitasi, dll).

## Potensi Penggunaan
Dataset ini dapat digunakan untuk berbagai keperluan penelitian dan analisis, antara lain:
1. **Analisis Hukum:** Studi tentang tren vonis hakim dalam perkara narkotika di Mataram.
2. **Kriminologi:** Analisis profil pelaku tindak pidana narkotika.
3. **Pemrosesan Bahasa Alami (NLP):** Ekstraksi informasi otomatis (seperti pasal yang dilanggar, lama hukuman) dari teks putusan hukum.

---
### Catatan Penting
> Data ini adalah dokumen publik yang diperoleh secara legal dari website resmi. Harap memperhatikan etika penggunaan data, terutama terkait privasi pihak-pihak yang tercantum dalam putusan meskipun sebagian identitas mungkin telah disamarkan oleh Mahkamah Agung.
